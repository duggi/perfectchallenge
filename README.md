## Get Started on OSX

```
> brew install rabbitmq
> brew install mongodb
> brew install nvm
> nvm use 0.10
> rabbitmq-server
> mongod --dbpath=/usr/local/var/mongodb/
> grunt
> localhost:3000
```


## Start App:

```
> rabbitmq-server
> mongod --dbpath=/usr/local/var/mongodb/
> grunt` or `node server.js
```
