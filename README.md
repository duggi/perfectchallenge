## Get Started on OSX

Use node *8.9.x*

```
npm install
brew install rabbitmq
brew install mongodb
rabbitmq-server
mongod --dbpath=/usr/local/var/mongodb/
grunt
open localhost:3000
```


## Start App:

```
rabbitmq-server
mongod --dbpath=/usr/local/var/mongodb/
grunt` or `node server.js
```
